

# 🔹 4. Chain-of-Thought Prompting

## 📖 Concept

Encourages the model to **reason step by step**.

---

## 🧾 Prompt

```
Solve step by step:

If 1 pen costs $2, how much for 4 pens?
```

---

## 💻 LLM Code

```python
response = client.chat.completions.create(
    model="gpt-5.3",
    messages=[
        {
            "role": "user",
            "content": "Solve step by step: If 1 pen costs $2, how much for 4 pens?"
        }
    ]
)

print(response.choices[0].message.content)
```

---

## ✅ Output

```
Step 1: Cost per pen = $2  
Step 2: Quantity = 4  
Step 3: Total = 2 × 4 = 8  

Answer: $8
```
