# 🚀 Phase 2 — Prompt Engineering (Core Skill)

This repository covers the **core concepts and practical techniques of Prompt Engineering**, a critical skill for working effectively with AI models like ChatGPT.

---

## 📌 What You’ll Learn

### 🔹 Zero-shot vs Few-shot Prompting

* **Zero-shot**: Asking the model to perform a task without providing examples.
* **Few-shot**: Supplying a few examples to guide the model toward better outputs.

---

### 🔹 System Prompts

System prompts define:

* Behavior of the AI
* Tone and style
* Constraints and rules

They help control how the model responds.

---

### 🔹 Chain-of-Thought Prompting

A technique that encourages the model to:

* Think step-by-step
* Improve reasoning accuracy
* Solve complex problems more effectively

---

### 🔹 Structured Outputs (JSON)

Learn how to:

* Format responses in JSON
* Make outputs machine-readable
* Integrate AI responses into applications

---

## 🛠️ Practice Tasks

Apply your knowledge by building prompts for:

### 📄 1. Summarization

Convert long text into concise summaries.

### 🔍 2. Information Extraction

Extract key data such as:

* Names
* Dates
* Important facts

### 🧠 3. Reasoning

Solve problems using logical step-by-step thinking.

---

## 🎯 Goal

By the end of this phase, you will be able to:

* Design effective prompts
* Improve AI response quality
* Build structured and reliable outputs
* Apply prompt techniques to real-world use cases
